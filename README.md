# 聚合搜索平台后端
基于SpringBoot+ElasticStack(+Vue3)的一站式聚合搜索平台。用户可以在同一个页面集中搜索出不同来源、不同类型的类型（比如文章、图片、用户、视频等），提升搜索体验。
